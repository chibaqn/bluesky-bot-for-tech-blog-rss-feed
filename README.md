## Set Up

Install TypeScript and Node.
```
npm i -g typescript
npm i -g ts-node
```

Install the dependencies.

```
npm ci
```

Copy the `example.env` file to `.env`.

```
cp example.env .env
```

Fill in `.env` with your Bluesky username and password.

## Running the script 

You can run the script locally: 

```
npx ts-node src/index.ts
```
